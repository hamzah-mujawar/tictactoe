# Tic Tac Toe written in C++ and Qt 
- Features a computer opponent that uses the minimax algorithm to choose its next move.

## Running the GUI 
- If not using NixOS or Nix you will need Qt6 and CMake to run the GUI 
- first `cd` into GUI and then `cmake -B build` and then `cmake --build build && ./build/tictactoe-gui` to run the GUI.

## Running the TUI 
- first `cd` into TUI and then `cmake -B build` and then `cmake --build build && ./build/tictactoe-tui` to run the TUI.
